本文对沁恒CH343(串口转USB)的供电方案做一个简要的介绍

下面是CH343的四个电源输入端：
VDD5：电源调节器的输入端
V3：内部电源调节器输出和内核及 USB 电源输入
VIO：I/O 端口电源输入端
VBUS：USB 总线的 VBUS 状态检测输入

其中VBUS不用多考虑，直接连USB的5V即可。

这个表看起来有些眼花缭乱，其实理清脉络还是很清晰的。

首先把握两个方向：使用的MCU主控的VDD多少是多少（5V、3.3V）。是否需要向USB取电（完全USB供电，部分使用，不使用USB供电）

当使用 5V 工作电压（大于 3.8V）时，VDD5 引脚输入外部 5V 电源（例如 USB 总线电源），由内部电源调节器于 V3 （需外接电容）引脚产生 3.3V 电源，用于 USB 收发器。
当使用 3.3V 或更低工作电压（小于 3.6V）时，V3 引脚应该与 VDD5 引脚相连接，同时输入外部的 3.3V 电源，V3 引脚仍需外接电源退耦电容。

需要注意的是如果使用内部V3的3.3V给MCU供电只能提供10mA的电流。
推荐双供电方案（见下图），即 VIO 与 MCU 共用电源VDD，VDD接USB的5V，V3直接接电容到地。
