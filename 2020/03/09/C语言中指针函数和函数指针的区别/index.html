C语言中指针函数和函数指针的区别
dadeju
  2020-03-09 14:07:42 2020-03-09 14:07

一、指针函数
指针函数的全称：返回值为指针值的函数。 形式通常是这样的： int *max(int n)
你完全可以简单理解为指针函数就只是一个函数只不过返回的是一个地址值，理所当然的你应该用一个指针来接受这个函数的返回值。

二、函数指针
函数指针的全称：指向函数的指针。 形式通常是这样的： int (*p)(int a,int b); 与前者只相差一个（）。
通常的用法： int (p)(int a,int b); //定义一个函数指针
int sum(int x,int y); //声明一个函数
p = sum; //将指针P指向函数sum
（p)(a,b); //调用指针P指向的sum函数
可以看到函数指针的的参数理所当然的应该与所指向的函数参数应该一致。
理解这两个概念并不是一件容易的事情，需要一些练习和犯一些错误。
